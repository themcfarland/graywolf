# AX.25 Connected-Mode Terminal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Operators can open a "Terminal" tab in the graywolf web UI, connect to a remote BBS or KISS-aware station over the radio using AX.25 v2.0/v2.2 connected mode, and interact through a polished xterm.js terminal pinned at 80×24. The full state machine, WebSocket bridge, and Svelte route ship together on a single feature branch behind a per-channel mode flag (`aprs` / `packet` / `aprs+packet`).

**Architecture:**

- **Backend.** New `pkg/ax25conn/` owns a hand-rolled AX.25 v2.0 LAPB data-link state machine (mod-8 first, mod-128 in Phase 3). One goroutine per session. Frames decoded by the existing modem subscribe via `pkg/app` ingress fanout; emitted frames go through `pkg/txgovernor` like every other TX source. A new `pkg/ax25termws/` glue maps a per-WebSocket connection to a session and bridges JSON envelopes ↔ session events. The existing `pkg/ax25/frame.go` extends to encode/decode I/S/U frames (UI-only today). New `Channel.Mode` enum gates beacon / digi / igate / messages TX per-channel; `MessagesConfig.TxChannel` migrates off `IGateConfig.TxChannel`.
- **Frontend.** New `web/src/routes/Terminal.svelte` route hosts an xterm.js terminal pinned at 80×24, surrounded by `@chrissnell/chonky-ui` chrome. Pre-connect form covers callsign / via path / saved profiles / advanced timer settings. APRS-only channel selection swaps to a read-only TNC2 raw-packet view backed by `pkg/packetlog`. Sidebar gets a Terminal entry with a `NotificationBadge` mirroring Messages. SauceCodePro Nerd Font self-hosted under `web/public/fonts/saucecodepro-nerd/`.
- **Persistence.** New tables: `ax25_session_profiles` (saved BBS shortcuts, pin to recents), `ax25_terminal_config` (singleton: theme/font/macros JSON/transcript-default), `ax25_transcript_sessions`, `ax25_transcript_entries`. New `messages_configs` singleton owns its own `tx_channel`. Migration copies values from `igate_configs.tx_channel` on first run.
- **Wire.** New endpoint `GET /api/ax25/terminal` upgrades to a WebSocket via `github.com/coder/websocket`. JSON envelopes (`{type, payload}`) where `payload` is a base64 byte slice for `data` types. One WS per active link, multi-tab via multiple WS connections.

**Tech stack:**

- Go 1.26.2, GORM/SQLite, `github.com/coder/websocket` v1.8+ (new dep)
- Svelte 5 runes, `svelte-spa-router`, `@chrissnell/chonky-ui ^0.4.0`, `@xterm/xterm 6.x` + `@xterm/addon-web-links` + `@xterm/addon-search` + `@xterm/addon-serialize` (new deps)
- Reference materials: AX.25 v2.0/v2.2 spec PDFs, K3NA 1988 paper, Linux kernel `net/ax25/` (v6.12), ax25-tools (linuxax25). See `pkg/ax25conn/CREDITS.md` (created in this plan) and the pre-research-derived behavioral cheat sheet at `.context/2026-05-01-ax25-lapb-behavioral-reference.md` — the cheat sheet is the **primary** source for Phase 1 transitions (every Phase 1 task cites a section).

## Scope, non-goals, and dev-tier phases

In scope:

- LAPB v2.0 (mod-8) outbound-only client, with v2.2 (mod-128/SABME/SREJ) negotiation as an opt-in advanced option.
- Per-channel mode flag with per-subsystem TX gating.
- WebSocket bridge, xterm.js Svelte route, multi-tab, sidebar unread badge, raw-packet APRS fallback view.
- Macro toolbar, transcript logging, recent-connections + pin, live telemetry side panel.
- `MessagesConfig.TxChannel` migration off iGate.

Out of scope (explicit non-goals):

- Inbound SABM acceptance (we DM unsolicited inbound). No BBS-host mode.
- Layer-3 (NET/ROM, ROSE, IP-over-AX.25), file-transfer protocols (YAPP, AutoBIN), v2.2 segmentation.
- Per-recipient channel routing for messages (separate followup).

Phase 0 ships first as its own merge to `main` — it is migration-heavy (touches `i_gate_configs`, beacon/digi/igate/messages TX gating) and independently shippable, and bundling it with the LAPB stack means a Phase-1 regression would block the Phase-0 admin improvements. Phases 1–3 land together in a second merge after Phase 0 has soaked one release cycle. This revises the brainstorm Q3 single-merge decision based on plan-review feedback.

| # | Phase | Merge | Outcome |
|---|---|---|---|
| 0 | Configuration plumbing | First merge to main | `Channel.Mode` migration, per-subsystem TX gates, `MessagesConfig` singleton + UI |
| 1 | LAPB core | Second merge (with 2/3) | `pkg/ax25conn/` state machine (mod-8), session manager, RX subscriber, TX governor wiring |
| 2 | WebSocket + terminal route | Second merge (with 1/3) | `/api/ax25/terminal` endpoint, `pkg/ax25termws/`, `Terminal.svelte`, xterm.js, multi-tab, pre-connect form |
| 3 | Polish | Second merge (with 1/2) | Mod-128 negotiation, telemetry side panel, macro toolbar, transcript logging + Transcripts subroute, recent + pin, raw-packet RX view |
| 4 | Integration + wiki | Final | On-air testing, wiki/handbook updates, final review |

```go
package configstore

import (
	"fmt"

	"gorm.io/gorm"
)

// migrateChannelsMode adds the channels.mode column (default 'aprs')
// when missing. SQLite ADD COLUMN with DEFAULT applies to existing
// rows in modern SQLite (3.35+). This migration runs in the
// pre-AutoMigrate phase so AutoMigrate sees the column exist with the
// right default and does not attempt a redundant ADD with NULL.
//
// See .context/2026-05-01-ax25-terminal-brainstorm.md §2.4 for why
// 'aprs' is the migrated-row default (preserve current behavior).
func migrateChannelsMode(tx *gorm.DB) error {
	hasCol, err := columnExists(tx, "channels", "mode")
	if err != nil {
		return fmt.Errorf("probe channels.mode: %w", err)
	}
	if hasCol {
		return nil
	}
	stmt := `ALTER TABLE channels ADD COLUMN mode TEXT NOT NULL DEFAULT 'aprs'`
	if err := tx.Exec(stmt).Error; err != nil {
		return fmt.Errorf("add channels.mode: %w", err)
	}
	return nil
}
```

If `columnExists` does not yet exist in the package, also add (only if not already present — check with `grep -n "func columnExists" graywolf/pkg/configstore/`):

```go
func columnExists(tx *gorm.DB, table, col string) (bool, error) {
	rows, err := tx.Raw(fmt.Sprintf(`PRAGMA table_info(%s)`, table)).Rows()
	if err != nil {
		return false, err
	}
	defer rows.Close()
	for rows.Next() {
		var cid int
		var name, ctype string
		var notnull, pk int
		var dflt any
		if err := rows.Scan(&cid, &name, &ctype, &notnull, &dflt, &pk); err != nil {
			return false, err
		}
		if name == col {
			return true, nil
		}
	}
	return false, nil
}
```

```go
package configstore

import "context"

// ChannelModeLookup is the small read-only surface the TX-gating
// subsystems (beacon, digipeater, igate, messages, ax25conn) consume
// to decide whether to permit a transmit on a given channel. The
// concrete *Store implements it; tests can substitute a fake.
type ChannelModeLookup interface {
	ModeForChannel(ctx context.Context, channelID uint32) (string, error)
}

// ModeForChannel returns the Mode column for the given channel id.
// Returns ChannelModeAPRS and a nil error when the row is missing,
// matching the migration default — TX subsystems treat "row not
// found" as the conservative APRS-only choice.
func (s *Store) ModeForChannel(ctx context.Context, channelID uint32) (string, error) {
	if channelID == 0 {
		return ChannelModeAPRS, nil
	}
	var mode string
	err := s.db.WithContext(ctx).
		Table("channels").
		Where("id = ?", channelID).
		Select("mode").
		Scan(&mode).Error
	if err != nil {
		return "", err
	}
	if mode == "" {
		return ChannelModeAPRS, nil
	}
	return mode, nil
}
```

```go
func (cfg *Config) ResolveTxChannel(ctx context.Context, candidates []uint32) uint32 {
```

```go
// migrateMessagesConfigCopyFromIgate creates messages_configs (id=1)
// and seeds TxChannel from i_gate_configs.tx_channel when the column
// is still present. Runs post-AutoMigrate so the messages_configs
// table exists. Skips the copy when a non-empty messages_configs
// already exists.
//
// LOAD-BEARING: this migration depends on i_gate_configs.tx_channel

// deleting this migration. See the matching comment on the
// IGateConfig.TxChannel struct field.
//
// See .context/2026-05-01-ax25-terminal-brainstorm.md §6.1.
func migrateMessagesConfigCopyFromIgate(tx *gorm.DB) error {
	var count int64
	if err := tx.Table("messages_configs").Count(&count).Error; err != nil {
		return fmt.Errorf("count messages_configs: %w", err)
	}
	if count > 0 {
		return nil
	}
	hasCol, err := columnExists(tx, "i_gate_configs", "tx_channel")
	if err != nil {
		return fmt.Errorf("probe i_gate_configs.tx_channel: %w", err)
	}

		if err := tx.Raw(`SELECT tx_channel FROM i_gate_configs WHERE id=1`).Scan(&igTx).Error; err != nil {
			// igate row may not exist on a fresh install — seed empty.
			igTx = 0
		}

		// Column removed by a later migration without first deleting
		// this one. Loud-log so operators notice the lost setting
		// rather than silently defaulting to 0.

		 VALUES (1, ?, datetime('now'), datetime('now'))`, igTx).Error; err != nil {
		return fmt.Errorf("seed messages_configs: %w", err)
	}
	return nil
}
```

# Phase 1 — LAPB core

This phase adds `pkg/ax25conn/` with a working AX.25 v2.0 (mod-8) state machine, plumbed into the existing RX fanout and TX governor. No UI, no WebSocket. Phase ships green when an integration test can drive the local LAPB session through SABM → CONNECTED → I-frame exchange → DISC → DISCONNECTED against a kernel-AX.25 peer (replayed from captured fixtures).

For every state-table behavioral edge case, code authors consult `.context/2026-05-01-ax25-lapb-behavioral-reference.md` first (which transcribes the kernel transitions with line citations); fall back to ax25-tools and the Linux kernel `net/ax25/` direct only when the cheat sheet is silent. Cite the source in the function's doc comment per `pkg/ax25conn/CREDITS.md` (Task 1.1).

**Kernel deviations we explicitly inherit (because Linux peers expect them):**

1. **No FRMR transmission.** Receipt of FRMR or any undecodable frame in CONNECTED/TIMER_RECOVERY tears the link to AWAITING_CONNECTION and re-SABMs. We do not originate FRMR. (See cheat sheet §4 — "the Linux kernel never sends FRMR".) Task 1.12 is an FRMR-receipt-only task; the encoder is documented in CREDITS as informational but not built.
2. **No SREJ.** REJ is the only out-of-order recovery, even on mod-128. (Cheat sheet §5.) Phase 3a.3 was originally "SREJ on mod-128"; it is now informational.
3. **Connection collision: state 1 stays in state 1 on inbound SABM.** No address comparison. Each side waits for its own SABM's UA. (Cheat sheet §3.)
4. **State 3 inbound SABM = graceful rebind.** Full reset (vs/vr/va=0, requeue ack queue), emit UA, re-arm T3. Not a "deliver buffered + accept new" operation. (Cheat sheet §1, state 3.)
5. **State 1 N2-exhaustion with mod-128 falls back to mod-8** rather than disconnecting. (Cheat sheet §1, state 1.)
6. **`condition` byte semantics**: `REJECT`, `ACK_PENDING`, `OWN_RX_BUSY`, `PEER_RX_BUSY` are bits in one struct field, zeroed wholesale at every link-establish, not on individual transitions. Replicate in `vars` struct (Task 1.4 update).

**Architectural rule from cheat sheet §10:** every state transition that could affect the sendable set must call the equivalent of `ax25_kick()` (drain pending I-frames if window has space and `!PEER_RX_BUSY`). Centralize that in a `Session.kick()` helper invoked at the tail of every `on<State>` handler that returned without exiting.

```go
package ax25conn

import "fmt"

type FrameKind uint8

const (
	FrameInvalid FrameKind = iota
	FrameI
	FrameRR
	FrameRNR
	FrameREJ
	FrameSREJ
	FrameSABM
	FrameSABME
	FrameDISC
	FrameDM
	FrameUA
	FrameFRMR
	FrameUI
	FrameXID
	FrameTEST
)

type Control struct {
	Kind FrameKind
	NS   uint8 // I-frames only (0..7 mod-8, 0..127 mod-128)
	NR   uint8 // I, RR, RNR, REJ, SREJ
	PF   bool  // poll/final
}

// Mod-8 control byte masks per AX.25 v2.0 §3.4.
const (
	mod8MaskNR = 0xE0
	mod8MaskPF = 0x10
	mod8MaskNS = 0x0E
)

// ParseControl decodes 1 byte (mod-8) or 2 bytes (mod-128) into a
// Control. Behavior follows AX.25 v2.0 §3.4 / v2.2 §4.2.
func ParseControl(b []byte, mod128 bool) (Control, error) {
	if mod128 {
		return parseControlMod128(b)
	}
	if len(b) < 1 {
		return Control{}, fmt.Errorf("ax25conn: short control")
	}
	c := Control{PF: b[0]&mod8MaskPF != 0}
	switch {
	case b[0]&0x01 == 0: // I-frame: low bit clear
		c.Kind = FrameI
		c.NS = (b[0] & mod8MaskNS) >> 1
		c.NR = (b[0] & mod8MaskNR) >> 5
	case b[0]&0x03 == 0x01: // S-frame: low two bits = 01
		c.NR = (b[0] & mod8MaskNR) >> 5
		switch b[0] & 0x0C { // bits 2-3 select S kind
		case 0x00:
			c.Kind = FrameRR
		case 0x04:
			c.Kind = FrameRNR
		case 0x08:
			c.Kind = FrameREJ
		case 0x0C:
			c.Kind = FrameSREJ
		}
	default: // U-frame: low two bits = 11
		switch b[0] &^ mod8MaskPF { // mask off P/F to get the kind
		case 0x2F:
			c.Kind = FrameSABM
		case 0x6F:
			c.Kind = FrameSABME
		case 0x43:
			c.Kind = FrameDISC
		case 0x0F:
			c.Kind = FrameDM
		case 0x63:
			c.Kind = FrameUA
		case 0x87:
			c.Kind = FrameFRMR
		case 0x03:
			c.Kind = FrameUI
		case 0xAF:
			c.Kind = FrameXID
		case 0xE3:
			c.Kind = FrameTEST
		default:
			return c, fmt.Errorf("ax25conn: unknown U-frame control 0x%02x", b[0])
		}
	}
	return c, nil
}

// EncodeControl renders a Control into 1 or 2 bytes. Caller selects
// modulo via the second arg; the function will error on out-of-range
// NS/NR for that modulo.
func EncodeControl(c Control, mod128 bool) ([]byte, error) {
	if mod128 {
		return encodeControlMod128(c)
	}
	var b byte
	if c.PF { b |= mod8MaskPF }
	switch c.Kind {
	case FrameI:
		if c.NS > 7 || c.NR > 7 { return nil, fmt.Errorf("ax25conn: mod-8 NS/NR overflow") }
		b |= (c.NR & 0x07) << 5
		b |= (c.NS & 0x07) << 1
		// low bit 0 already
	case FrameRR:   b |= ((c.NR & 0x07) << 5) | 0x01
	case FrameRNR:  b |= ((c.NR & 0x07) << 5) | 0x05
	case FrameREJ:  b |= ((c.NR & 0x07) << 5) | 0x09
	case FrameSREJ: b |= ((c.NR & 0x07) << 5) | 0x0D
	case FrameSABM: b |= 0x2F
	case FrameSABME:b |= 0x6F
	case FrameDISC: b |= 0x43
	case FrameDM:   b |= 0x0F
	case FrameUA:   b |= 0x63
	case FrameFRMR: b |= 0x87
	case FrameUI:   b |= 0x03
	case FrameXID:  b |= 0xAF
	case FrameTEST: b |= 0xE3
	default:
		return nil, fmt.Errorf("ax25conn: cannot encode FrameKind %v", c.Kind)
	}
	return []byte{b}, nil
}

}
func encodeControlMod128(c Control) ([]byte, error) {

}
```

```go
package ax25conn

import (
	"fmt"

	"github.com/chrissnell/graywolf/pkg/ax25"
)

// Frame is a connected-mode AX.25 frame. The address layout is
// identical to UI frames; only the control field and presence of
// PID/Info differ.
type Frame struct {
	Source, Dest ax25.Address
	Path         []ax25.Address
	Control      Control
	PID          byte // I and UI only
	Info         []byte
	IsCommand    bool // direwolf-style C-bit polarity (dest C set, source C clear)
	Mod128       bool
}

// hasInfo reports whether the frame's Control.Kind carries an info
// field (I, UI, FRMR, XID, TEST). Other kinds are control-only.
func (f *Frame) hasInfo() bool {
	switch f.Control.Kind {
	case FrameI, FrameUI, FrameFRMR, FrameXID, FrameTEST:
		return true
	}
	return false
}

// hasPID reports whether the frame carries a PID byte (I and UI only).
func (f *Frame) hasPID() bool {
	return f.Control.Kind == FrameI || f.Control.Kind == FrameUI
}

// Encode emits the wire bytes (no FCS — modem appends).
func (f *Frame) Encode() ([]byte, error) {
	if len(f.Path) > ax25.MaxRepeaters {
		return nil, fmt.Errorf("ax25conn: path length %d > %d", len(f.Path), ax25.MaxRepeaters)
	}
	addrBytes, err := ax25.EncodeAddressBlock(f.Source, f.Dest, f.Path, f.IsCommand)
	if err != nil { return nil, err }
	ctl, err := EncodeControl(f.Control, f.Mod128)
	if err != nil { return nil, err }
	out := make([]byte, 0, len(addrBytes)+len(ctl)+1+len(f.Info))
	out = append(out, addrBytes...)
	out = append(out, ctl...)
	if f.hasPID() {
		out = append(out, f.PID)
	}
	if f.hasInfo() {
		out = append(out, f.Info...)
	}
	return out, nil
}

// Decode parses connected-mode frame bytes.
func Decode(raw []byte, mod128 bool) (*Frame, error) {
	hdr, err := ax25.DecodeAddressBlock(raw)
	if err != nil { return nil, err }
	off := hdr.AddrLen
	ctlSize := 1
	if mod128 { ctlSize = 2 }
	if len(raw) < off+ctlSize {
		return nil, fmt.Errorf("ax25conn: missing control field")
	}
	c, err := ParseControl(raw[off:off+ctlSize], mod128)
	if err != nil { return nil, err }
	off += ctlSize
	f := &Frame{
		Source:    hdr.Source,
		Dest:      hdr.Dest,
		Path:      hdr.Path,
		Control:   c,
		IsCommand: hdr.IsCommand,
		Mod128:    mod128,
	}
	if f.hasPID() {
		if off >= len(raw) { return nil, fmt.Errorf("ax25conn: missing PID") }
		f.PID = raw[off]
		off++
	}
	if f.hasInfo() {
		f.Info = append([]byte(nil), raw[off:]...)
	}
	return f, nil
}
```

```go
	case StateConnected:           return "CONNECTED"

// vars holds per-session sequence and flag state. Mutated only from
// the session goroutine; readers go through atomic snapshot getters
// when used outside that goroutine.
//
// The Cond bitfield mirrors the kernel's `ax25->condition` byte
// (include/net/ax25.h:60-64). The kernel zeroes the whole byte at
// link-establish time (ax25_std_subr.c:37); replicate via
// `s.v.Cond = 0` in establish-data-link, not via per-bit clears.
type vars struct {
	VS, VR, VA uint8         // send / receive / ack sequence numbers
	N2Count    int           // retry count (cmp against cfg.N2)
	Cond       Condition     // bitfield, see below
	RTT        time.Duration // EWMA, seeded at T1/2 on link establish
	T1Started  time.Time     // wall time of last T1 reset; used by RTT calc
}

type Condition uint8

const (
	// CondACKPending — we received an in-order I-frame without P=1
	// and have not yet emitted an RR/RNR; T2 will fire if no
	// piggyback opportunity arrives. Kernel AX25_COND_ACK_PENDING.
	CondACKPending Condition = 1 << iota
	// CondReject — we sent REJ for an out-of-order I-frame and are
	// waiting for the missing N(S). Don't send another REJ until
	// CondReject clears. Kernel AX25_COND_REJECT.
	CondReject
	// CondPeerRxBusy — peer sent RNR; suspend our I-frame TX.
	// Kernel AX25_COND_PEER_RX_BUSY.
	CondPeerRxBusy
	// CondOwnRxBusy — our receive buffer is full; we're sending RNR
	// in lieu of RR. Cleared by the heartbeat once buffer drains.
	// Kernel AX25_COND_OWN_RX_BUSY.
	CondOwnRxBusy
)

func (c Condition) Has(b Condition) bool { return c&b != 0 }
func (c *Condition) Set(b Condition)     { *c |= b }
func (c *Condition) Clear(b Condition)   { *c &^= b }

// modulus returns 8 or 128 depending on Mod128.
func (s *Session) modulus() int {
	if s.cfg.Mod128 { return 128 }
	return 8
```

```go
package ax25conn

import (
	"sync"
	"time"
)

// Clock abstracts time for tests.
type Clock interface {
	Now() time.Time
	NewTimer(d time.Duration) *time.Timer
}

type realClock struct{}
func (realClock) Now() time.Time            { return time.Now() }
func (realClock) NewTimer(d time.Duration) *time.Timer { return time.NewTimer(d) }

// timer wraps a single duration with reset semantics matching AX.25
// v2.2 §4.3.5: reset stops a running timer and starts it from zero.
type timer struct {
	mu    sync.Mutex
	clk   Clock
	d     time.Duration
	t     *time.Timer
	on    bool
	fire  func()
}

func newTimer(clk Clock, d time.Duration, fire func()) *timer {
	return &timer{clk: clk, d: d, fire: fire}
}

func (t *timer) reset() {
	t.mu.Lock()
	defer t.mu.Unlock()
	t.stopLocked()
	t.t = t.clk.NewTimer(t.d)
	t.on = true
	go t.wait(t.t)
}

func (t *timer) stop() {
	t.mu.Lock()
	defer t.mu.Unlock()
	t.stopLocked()
}

func (t *timer) stopLocked() {
	if !t.on { return }
	t.on = false
	if t.t != nil { t.t.Stop() }
}

func (t *timer) wait(timer *time.Timer) {
	<-timer.C
	t.mu.Lock()
	if !t.on || t.t != timer {
		t.mu.Unlock()
		return
	}
	t.on = false
	t.mu.Unlock()
	t.fire()
}
```

```go
func (s *Session) heartbeatTick() {
```

```go
// Pending-timer bits. Timer goroutines OR these into pendingTimers
// atomically; the run loop drains them after each channel receive.
// This makes timer events guaranteed-delivery without unbounding the
// frame channel — a dropped T1 expiry would silently hang the link in
// TIMER_RECOVERY without ever reaching N2 retries.
const (
	pendT1 uint32 = 1 << iota
	pendT2
	pendT3
	pendHB
)

type Session struct {
	cfg     SessionConfig
	in      chan Event
	state   State
	v       vars
	t1, t2, t3, hb *timer
	pendingTimers atomic.Uint32 // OR-set by timer goroutines, drained by run loop
	wakeup        chan struct{} // 1-slot signal that pendingTimers changed
	stats   LinkStats
	// I-frame retransmit buffer keyed by NS for retransmission on REJ.
	pending [128]*Frame
}

func NewSession(cfg SessionConfig) (*Session, error) {
	cfg.applyDefaults()
	if cfg.TxSink == nil {
		return nil, fmt.Errorf("ax25conn: TxSink required")
	}
	if cfg.Local.Call == "" || cfg.Peer.Call == "" {
		return nil, fmt.Errorf("ax25conn: Local and Peer required")
	}
	s := &Session{
		cfg:    cfg,
		in:     make(chan Event, 32),
		wakeup: make(chan struct{}, 1),
		state:  StateDisconnected,
	}
	s.t1 = newTimer(cfg.Clock, cfg.T1, func() { s.signalTimer(pendT1) })
	s.t2 = newTimer(cfg.Clock, cfg.T2, func() { s.signalTimer(pendT2) })
	s.t3 = newTimer(cfg.Clock, cfg.T3, func() { s.signalTimer(pendT3) })
	s.hb = newTimer(cfg.Clock, cfg.Heartbeat, func() { s.signalTimer(pendHB) })

// enqueue routes a non-timer event onto the bounded channel. Drop-on-
// full is acceptable here: EventFrameRX bursts are recoverable via
// LAPB retransmission; EventDataTX losses cause operator-visible UI
// errors that prompt a retry. Timer events do NOT use this path —
// see signalTimer.
func (s *Session) enqueue(e Event) {
	select {
	case s.in <- e:
	default:
		s.cfg.Logger.Warn("ax25conn: dropping event; session input full",
			"kind", e.Kind, "peer", s.cfg.Peer.String())
	}
}

// signalTimer ORs a timer-pending bit and wakes the run loop. Idempotent
// across multiple firings before the loop drains; the run loop processes
// each kind of timer expiry at most once per drain cycle, matching
// kernel ax25_timer.c semantics.
func (s *Session) signalTimer(bit uint32) {
	s.pendingTimers.Or(bit)
	select {
	case s.wakeup <- struct{}{}:
	default:
	}
}

// Run blocks until ctx is cancelled or the session reaches a terminal
// state. Manager invokes Run in a goroutine. Each iteration drains
// pending timer bits before reading the channel so timer events never
// starve under RX bursts.
func (s *Session) Run(ctx context.Context) {
	defer s.cleanup()
	for {
		// Drain pending timer events first — guaranteed delivery.
		if bits := s.pendingTimers.Swap(0); bits != 0 {
			if bits&pendT1 != 0 && !s.handle(ctx, Event{Kind: EventT1Expiry}) { return }
			if bits&pendT2 != 0 && !s.handle(ctx, Event{Kind: EventT2Expiry}) { return }
			if bits&pendT3 != 0 && !s.handle(ctx, Event{Kind: EventT3Expiry}) { return }
			if bits&pendHB != 0 && !s.handle(ctx, Event{Kind: EventHeartbeat}) { return }
			continue
		}
		select {
		case <-ctx.Done():
			return
		case <-s.wakeup:
			continue // loop back, drain pendingTimers
		case ev := <-s.in:
			if !s.handle(ctx, ev) {
				return
			}
		}
	}
}

// handle dispatches by current state. Returns false when the session
// should exit.
func (s *Session) handle(ctx context.Context, ev Event) bool {
	switch s.state {
	case StateDisconnected:
		return s.onDisconnected(ctx, ev)
	case StateAwaitingConnection:
		return s.onAwaitingConnection(ctx, ev)
	case StateConnected:
		return s.onConnected(ctx, ev)
	case StateTimerRecovery:
		return s.onTimerRecovery(ctx, ev)
	case StateAwaitingRelease:
		return s.onAwaitingRelease(ctx, ev)
	}
	return false
}

func (s *Session) cleanup() {
```

```go
// onDisconnected — state 0. Behavior matches kernel ax25_in.c:317-427
// dispatch (the kernel has no state-0 input handler; ax25_rcv() decides
// directly). Cheat sheet §1 state 0.
func (s *Session) onDisconnected(ctx context.Context, ev Event) bool {
	switch ev.Kind {
	case EventConnect:
		// Equivalent of ax25_std_establish_data_link
		// (ax25_std_subr.c:35-50): zero condition, n2count=0, TX
		// SABM(P=1, cmd), start T1, seed RTT at T1/2.
		s.v = vars{RTT: s.cfg.T1 / 2}
		s.stats = LinkStats{State: StateAwaitingConnection}
		s.sendSABM(s.cfg.Mod128)
		s.t1.reset()
		s.setState(StateAwaitingConnection)
		return true
	case EventFrameRX:
		// Outbound-only client (brainstorm Q4). Reject inbound SABM/SABME
		// with DM(F=p, rsp). Silently drop DM (kernel "Never reply to a
		// DM" — ax25_in.c:323-326). Drop everything else silently.
		if ev.Frame == nil { return true }
		switch ev.Frame.Control.Kind {
		case FrameSABM, FrameSABME:
			s.sendDM(ev.Frame.Control.PF)
		}
		return true
	case EventShutdown:
		return false
	}
	return true
}

// sendSABM emits SABM (mod-8) or SABME (mod-128) with P=1, cmd. The
// kernel chooses by ax25->modulus in ax25_std_subr.c:35-50. We accept
// the modulus arg explicitly so the state-1 N2-exhaustion fall-back
// (cheat sheet §1 state 1) can downgrade in-place.
func (s *Session) sendSABM(mod128 bool) {
	kind := FrameSABM
	if mod128 { kind = FrameSABME }
	f := &Frame{
		Source: s.cfg.Local, Dest: s.cfg.Peer, Path: s.cfg.Path,
		Control: Control{Kind: kind, PF: true},
		IsCommand: true, Mod128: false, // header is always mod-8 at TX-encode
	}
	s.submit(f)
}

func (s *Session) sendDM(pf bool) {
	f := &Frame{
		Source: s.cfg.Local, Dest: s.cfg.Peer, Path: s.cfg.Path,
		Control: Control{Kind: FrameDM, PF: pf},
		IsCommand: false, Mod128: false,
	}
	s.submit(f)
}

// submit hands a connected-mode frame to the TX governor with our
// priority. Connected-mode frames are passed as `*ax25.Frame` so the
// governor's hooks/logging/dedup machinery applies uniformly. Dedup is
// bypassed via SkipDedup=true because identical bytes legitimately
// retransmit on T1 expiry.
func (s *Session) submit(f *Frame) {
	wrapper, err := f.ToAX25Frame() // see Task 1.7b for ax25.Frame extension
	if err != nil {
		s.cfg.Logger.Warn("ax25conn: encode failed", "err", err); return
	}
	src := txgovernor.SubmitSource{
		Kind: "ax25conn", Priority: ax25.PriorityAX25Conn, SkipDedup: true,
	}
	if err := s.cfg.TxSink.SubmitRaw(context.Background(), s.cfg.Channel, wrapper, src); err != nil {
		s.cfg.Logger.Warn("ax25conn: tx submit failed", "err", err)
	}
	s.stats.FramesTX++
}
```

```go
// onAwaitingConnection — state 1, "SABM sent". Behavior matches
// ax25_std_state1_machine in net/ax25/ax25_std_in.c:39-96 of Linux
// v6.12. Kernel deviation from spec: inbound SABM does not transition
// us to state 3 — we emit UA and wait for our own SABM's UA per the
// kernel pattern. See cheat sheet §3 "Connection collision".
func (s *Session) onAwaitingConnection(ctx context.Context, ev Event) bool {
	switch ev.Kind {
	case EventFrameRX:
		f := ev.Frame
		if f == nil { return true }
		switch f.Control.Kind {
		case FrameSABM:
			if !f.IsCommand { return true } // malformed: drop
			// Peer sent SABM (collision or simultaneous-open). Match
			// kernel: emit UA, set modulus=8, stay in state 1.
			s.cfg.Mod128 = false
			s.sendUA(f.Control.PF)
		case FrameSABME:
			if !f.IsCommand { return true } // malformed: drop
			// Mod-128 peer SABM. Match kernel: UA, set modulus=128.
			s.cfg.Mod128 = true
			s.sendUA(f.Control.PF)
		case FrameDISC:
			if !f.IsCommand { return true } // malformed: drop
			s.sendDM(f.Control.PF)
		case FrameUA:
			if !f.Control.PF { return true } // F=0 ignored (kernel)
			s.t1.stop()
			s.calcRTT()
			s.v.N2Count = 0
			s.v.VS, s.v.VR, s.v.VA = 0, 0, 0
			s.t3.reset()
			s.setState(StateConnected)
		case FrameDM:
			if !f.Control.PF { return true } // F=0 ignored
			if s.cfg.Mod128 {
				// Kernel ax25_std_in.c:84-87: downgrade to mod-8,
				// keep T1 running, retry as SABM on next T1 expiry.
				s.cfg.Mod128 = false
				s.cfg.Window = DefaultWindowMod8
				return true
			}
			s.t1.stop()
			s.emit(OutEvent{Kind: OutError, ErrCode: "peer-rejected",
				ErrMsg: "DM during link setup"})
			s.setState(StateDisconnected)
		}
	case EventT1Expiry:
		if s.v.N2Count >= s.cfg.N2 {
			if s.cfg.Mod128 {
				// Kernel ax25_std_timer.c:128-133: auto-fall-back to
				// mod-8 with a fresh N2 budget.
				s.cfg.Mod128 = false
				s.cfg.Window = DefaultWindowMod8
				s.v.N2Count = 0
				s.sendSABM(false)
				s.t1.reset()
				return true
			}
			s.emit(OutEvent{Kind: OutError, ErrCode: "link-setup-timeout",
				ErrMsg: "no response to SABM after N2 retries"})
			s.setState(StateDisconnected)
			return true
		}
		s.v.N2Count++
		s.sendSABM(s.cfg.Mod128)
		s.t1.reset()
	case EventDisconnect, EventAbort:
		// Kernel af_ax25.c:1010-1018 — close in state 1 sends DISC and
		// transitions through ax25_disconnect to state 0. We model the
		// intermediate AWAITING_RELEASE state for symmetry.
		s.sendDISC()
		s.v.N2Count = 0
		s.t1.reset()
		s.setState(StateAwaitingRelease)
	case EventShutdown:
		return false
	}
	return true
}

// calcRTT folds the latest T1 measurement into the RTT EWMA per
// ax25_calculate_rtt (net/ax25/ax25_subr.c:220-258). Sampled only on
// first-shot success (n2count==0); retransmits don't poison the
// estimate.
func (s *Session) calcRTT() {
	if s.v.N2Count != 0 || s.v.T1Started.IsZero() { return }
	measured := s.cfg.Clock.Now().Sub(s.v.T1Started)
	if measured < 0 { return }
	if s.v.RTT == 0 { s.v.RTT = s.cfg.T1 / 2 }
	s.v.RTT = (9*s.v.RTT + measured) / 10
	if s.v.RTT < RTTClampLo { s.v.RTT = RTTClampLo }
	if s.v.RTT > RTTClampHi { s.v.RTT = RTTClampHi }
}
```

```markdown
deviation from spec §4.3.3.9: the kernel never sends FRMR

or any frame `ax25_decode()` flags as illegal, we tear the link to
state 1 and re-SABM via `establishDataLink()`. This both matches the
kernel and is simpler than maintaining an FRMR info-byte encoder
that no Linux peer would ever accept anyway.

If a future Phase wants spec-correct FRMR transmission, the v2.2
info-field layout is documented in
.context/2026-05-01-ax25-lapb-behavioral-reference.md §4 and AX.25
v2.2 §4.3.3.9. Note: kernel peers will tear the link down on FRMR
receipt regardless of info-field validity, so spec-correctness gains
nothing on the wire.
```

```go
package ax25conn

import (
	"context"
	"errors"
	"fmt"
	"log/slog"
	"sync"
	"sync/atomic"

	"github.com/chrissnell/graywolf/pkg/configstore"
)

type ManagerConfig struct {
	TxSink         RawTxSink
	ChannelModes   configstore.ChannelModeLookup
	Logger         *slog.Logger
	MaxPerOperator int // default 4 (per-process; resets across restart)
	MaxTotal       int // default 64 (per-process; resets across restart)
}

// Caps are process-local — they limit concurrent live LAPB sessions
// inside one graywolf instance. After a graywolf restart, prior sessions
// are gone and operators may reconnect freely. The same operator across
// multiple browser sessions or API tokens accumulates against the cap
// inside this process. This is intentional and matches the kernel
// AX.25 stack's per-process accounting; no cross-process persistence.

type Manager struct {
	cfg     ManagerConfig
	mu      sync.Mutex
	byKey   map[sessionKey]*managedSession
	nextID  atomic.Uint64
	closed  bool
}

// sessionKey routes inbound LAPB frames to the owning Session. Keying
// is intentionally `(channel, local, peer)` — *not* the via path —
// because:
//   1. AX.25 v2.x semantics: a (local, peer) pair has at most one
//      logical link. Two sessions over different via paths to the same
//      peer would race.
//   2. Inbound frames carry the *forward* path the digipeaters
//      already consumed, while the Session was opened with the *outbound*
//      via list. Path-fingerprint comparison would require the
//      inbound-path normalization that Task 1.13b adds for
//      identification, but it is not used for routing.
//
// Manager.Open returns an error when a second session for the same
// triple is requested. Operators wanting to reach the same peer over
// a different path must close the first session first.
type sessionKey struct{ Channel uint32; Local, Peer string }

type managedSession struct {
	id     uint64
	s      *Session
	cancel context.CancelFunc
	op     string // operator who opened (for cap enforcement)
}

func NewManager(cfg ManagerConfig) *Manager {
	if cfg.MaxPerOperator == 0 { cfg.MaxPerOperator = 4 }
	if cfg.MaxTotal == 0       { cfg.MaxTotal = 64 }
	if cfg.Logger == nil       { cfg.Logger = slog.Default() }
	return &Manager{cfg: cfg, byKey: make(map[sessionKey]*managedSession)}
}

// Open creates and starts a new session bound to (channel, local, peer).
// Returns the session id and the input handle for the bridge to push
// EventDataTX/EventDisconnect events.
func (m *Manager) Open(scfg SessionConfig, operator string) (uint64, *Session, error) {
	m.mu.Lock()
	defer m.mu.Unlock()
	if m.closed { return 0, nil, errors.New("ax25conn: manager closed") }
	// Per-channel-mode gate.
	if m.cfg.ChannelModes != nil {
		mode, _ := m.cfg.ChannelModes.ModeForChannel(context.Background(), scfg.Channel)
		if mode == configstore.ChannelModeAPRS {
			return 0, nil, fmt.Errorf("ax25conn: channel %d is APRS-only", scfg.Channel)
		}
	}
	// Cap.
	if len(m.byKey) >= m.cfg.MaxTotal {
		return 0, nil, errors.New("ax25conn: max total sessions reached")
	}
	count := 0
	for _, ms := range m.byKey {
		if ms.op == operator { count++ }
	}
	if count >= m.cfg.MaxPerOperator {
		return 0, nil, errors.New("ax25conn: per-operator session cap reached")
	}
	key := sessionKey{Channel: scfg.Channel, Local: scfg.Local.String(), Peer: scfg.Peer.String()}
	if _, exists := m.byKey[key]; exists {
		return 0, nil, errors.New("ax25conn: session already exists for this triple")
	}
	scfg.TxSink = m.cfg.TxSink // wired through the manager
	s, err := NewSession(scfg)
	if err != nil { return 0, nil, err }
	id := m.nextID.Add(1)
	ctx, cancel := context.WithCancel(context.Background())
	ms := &managedSession{id: id, s: s, cancel: cancel, op: operator}
	m.byKey[key] = ms
	go func() {
		defer m.remove(key)
		s.Run(ctx)
	}()
	return id, s, nil
}

func (m *Manager) remove(k sessionKey) {
	m.mu.Lock()
	defer m.mu.Unlock()
	delete(m.byKey, k)
}

// Dispatch routes a non-UI frame to the matching session. Caller is the
// pkg/app rxfanout; per the brainstorm §4.1, we already know the frame
// is non-UI.
func (m *Manager) Dispatch(channel uint32, f *Frame) {
	if f == nil { return }
	key := sessionKey{Channel: channel, Local: f.Dest.String(), Peer: f.Source.String()}
	m.mu.Lock()
	ms := m.byKey[key]
	m.mu.Unlock()
	if ms == nil { return }
	ms.s.Submit(Event{Kind: EventFrameRX, Frame: f})
}

func (m *Manager) Close() {
	m.mu.Lock()
	defer m.mu.Unlock()
	m.closed = true
	for _, ms := range m.byKey {
		ms.cancel()
	}
}

// Run blocks until ctx is done; provided for symmetric shape with
// other graywolf services. Sessions are managed via Open; Run is a
// no-op loop that just waits for shutdown.
func (m *Manager) Run(ctx context.Context) {
	<-ctx.Done()
	m.Close()
}
```

```go
package ax25termws

import (
	"context"
	"errors"
	"log/slog"

	"github.com/chrissnell/graywolf/pkg/ax25"
	"github.com/chrissnell/graywolf/pkg/ax25conn"
)

type BridgeConfig struct {
	Manager  *ax25conn.Manager
	Logger   *slog.Logger
	Operator string // current authenticated user
	// Ctx scopes the bridge's lifetime. Blocking sends on Out unblock
	// when Ctx is done, preventing the session goroutine from leaking
	// after the WebSocket closes.
	Ctx context.Context
	// Out is the channel the bridge fills with outbound envelopes; the
	// WebSocket handler drains it.
	Out chan<- Envelope
}

type Bridge struct {
	cfg     BridgeConfig
	session *ax25conn.Session
	id      uint64
}

func New(cfg BridgeConfig) *Bridge { return &Bridge{cfg: cfg} }

// Handle dispatches one inbound envelope to the appropriate side
// effect. Returns an error if the message cannot be processed.
func (b *Bridge) Handle(ctx context.Context, env Envelope) error {
	switch env.Kind {
	case KindConnect:
		if b.session != nil { return errors.New("session already open") }
		if env.Connect == nil { return errors.New("connect: missing args") }
		local, err := ax25.ParseAddress(formatAddr(env.Connect.LocalCall, env.Connect.LocalSSID))
		if err != nil { return err }
		peer, err := ax25.ParseAddress(formatAddr(env.Connect.DestCall, env.Connect.DestSSID))
		if err != nil { return err }
		path := make([]ax25.Address, 0, len(env.Connect.Via))
		for _, p := range env.Connect.Via {
			a, err := ax25.ParseAddress(p); if err != nil { return err }
			path = append(path, a)
		}
		scfg := ax25conn.SessionConfig{
			Local: local, Peer: peer, Path: path,
			Channel: env.Connect.ChannelID,
			Mod128:  env.Connect.Mod128,
			N2:      env.Connect.N2,
			Paclen:  env.Connect.Paclen,
			Window:  env.Connect.Maxframe,
			Logger:  b.cfg.Logger,
			Observer: b.observe,
		}
		if env.Connect.T1MS > 0 { scfg.T1 = millis(env.Connect.T1MS) }
		if env.Connect.T2MS > 0 { scfg.T2 = millis(env.Connect.T2MS) }
		if env.Connect.T3MS > 0 { scfg.T3 = millis(env.Connect.T3MS) }
		id, sess, err := b.cfg.Manager.Open(scfg, b.cfg.Operator)
		if err != nil { return err }
		b.session, b.id = sess, id
		sess.Submit(ax25conn.Event{Kind: ax25conn.EventConnect})
	case KindData:
		if b.session == nil { return errors.New("not connected") }
		b.session.Submit(ax25conn.Event{Kind: ax25conn.EventDataTX, Data: env.Data})
	case KindDisconnect:
		if b.session != nil {
			b.session.Submit(ax25conn.Event{Kind: ax25conn.EventDisconnect})
		}
	case KindAbort:
		if b.session != nil {
			b.session.Submit(ax25conn.Event{Kind: ax25conn.EventAbort})
		}
	default:
		return errors.New("unknown kind: " + string(env.Kind))
	}
	return nil
}

// observe is the session.Observer callback. Translates OutEvent into
// envelopes pushed onto cfg.Out. Send semantics differ by kind:
//
//   - OutDataRX uses BLOCKING send (with ctx cancellation). Operator-
//     visible byte stream from the BBS — LAPB has already ack'd the
//     I-frame to the peer, so a dropped envelope is unrecoverable.
//     Back-pressure must propagate into the session: if the bridge
//     can't drain, the session goroutine blocks here, which delays
//     `kick()`/RR generation and naturally throttles the peer via the
//     LAPB window. This is the desired behavior.
//
//   - OutStateChange / OutLinkStats / OutError use NON-BLOCKING send
//     with drop-on-full + warn log. These are idempotent snapshots
//     (latest state/stats supersede earlier ones) and safe to drop.
func (b *Bridge) observe(ev ax25conn.OutEvent) {
	var env Envelope
	switch ev.Kind {
	case ax25conn.OutStateChange:
		env = Envelope{Kind: KindState, State: &StatePayload{Name: ev.State.String()}}
	case ax25conn.OutDataRX:
		env = Envelope{Kind: KindDataRX, Data: ev.Data}
		// Blocking send for data; ctx.Done unblocks on session shutdown.
		select {
		case b.cfg.Out <- env:
		case <-b.cfg.Ctx.Done():
		}
		return
	case ax25conn.OutLinkStats:
		env = Envelope{Kind: KindLinkStats, Stats: linkStatsToPayload(ev.Stats)}
	case ax25conn.OutError:
		env = Envelope{Kind: KindError, Error: &ErrorPayload{Code: ev.ErrCode, Message: ev.ErrMsg}}
	}
	select {
	case b.cfg.Out <- env:
	default:
		b.cfg.Logger.Warn("ax25termws: out buffer full; dropping envelope",
			"kind", env.Kind)
	}
}
```

## Task 2.10: Single-session client store

**Files:**
- Create: `graywolf/web/src/lib/terminal/session.svelte.js`

- [ ] **Step 1: Implement** the store: opens a WebSocket, pushes `connect`, exposes reactive state (`status`, `peer`, `via`, `unread`, `stats`, `errorMessage`) and methods (`sendData(bytes)`, `disconnect()`, `abort()`, `close()`). Receive messages by switching on `env.kind`.

```js
import { ANSI_PALETTE } from './palette.ts';
import { encodeData } from './envelope.js';

export function createSession(initial) {
  let ws = null;
  const state = $state({
    id: crypto.randomUUID(),
    peer: initial.dest,
    via: initial.via ?? [],
    channelId: initial.channelId,
    status: 'connecting', // connecting | connected | disconnected | error
    unreadBytes: 0,
    stats: null,
    errorMessage: null,
  });

  function open() {
    ws = new WebSocket(`ws://${location.host}/api/ax25/terminal`);
    ws.binaryType = 'arraybuffer';
    ws.onopen = () => ws.send(JSON.stringify({ kind: 'connect', connect: initial }));
    ws.onmessage = (ev) => {
      const env = JSON.parse(ev.data);
      handle(env);
    };
    ws.onerror = (e) => { state.errorMessage = 'WebSocket error'; };
    ws.onclose = () => { state.status = 'disconnected'; };
  }

  function handle(env) {
    switch (env.kind) {
      case 'state':
        state.status = env.state.name === 'CONNECTED' ? 'connected'
                     : env.state.name === 'DISCONNECTED' ? 'disconnected'
                     : 'connecting';
        break;
      case 'data_rx': {
        const bytes = atob(env.data);
        state.onDataRX?.(bytes);
        if (!state.focused) state.unreadBytes += bytes.length;
        break;
      }
      case 'link_stats': state.stats = env.stats; break;
      case 'error':      state.errorMessage = env.error.message; break;
    }
  }

  function sendData(bytes) {
    if (!ws || ws.readyState !== WebSocket.OPEN) return;
    ws.send(JSON.stringify({ kind: 'data', data: encodeData(bytes) }));
  }
  function disconnect() { ws?.send(JSON.stringify({ kind: 'disconnect' })); }
  function abort()      { ws?.send(JSON.stringify({ kind: 'abort' })); }
  function close()      { ws?.close(); }

  open();
  return { state, sendData, disconnect, abort, close };
}
```
