Station Callsign

One callsign for the whole station

Your station callsign is the single source of truth that the iGate, digipeater, beacons, and messenger use by default. Setting it once here means you don’t have to retype it (and re-typo it) on every other page.

Open Settings → Station Callsign, type your callsign with the SSID you want to use as your default (e.g. NW5W-5 for a fixed digi/iGate, NW5W-9 for a tracker), and click Save. Every page that defaults to the station callsign reads it back as (inherited).

Where it’s used

  • iGate — APRS-IS login.
  • Beacons — default source callsign on every beacon. You can still override per beacon if a tracker needs a different SSID.
  • Digipeater — the callsign your digi transmits under, and the one used for preemptive digi when a packet’s path explicitly names you. Override if you want the digi to wear a different SSID than the rest of the station.
  • Messages — the From callsign on outbound DMs and tactical posts.
⚠

If no station callsign is set, sending a message returns a clear error pointing you back here. Set this before you put your station on the air.
