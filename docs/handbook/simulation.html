Simulation Mode

Run the full station with the transmit muted

Simulation mode lets you exercise every part of graywolf — digipeater, iGate, beacons, messaging — without keying the radio. PTT stays down and audio out is suppressed; everything else (RX, packet log, beacons, gating decisions, web UI updates) runs normally. It’s the right switch to flip when you’re tuning rules and want to see what would have transmitted before you actually let it transmit.

Turning it on

Open Settings → Simulation, tick Enable simulation mode, and click Apply. The setting takes effect immediately and survives restarts — remember to turn it off again when you’re ready to go live.

What changes

  • Outbound packets are processed end-to-end (CSMA, dedup, rate limit, third-party wrap, …) and logged, but never modulated to the radio.
  • iGate IS→RF gating still runs every rule and records what it would have keyed, so you can validate a server filter and rule set against real APRS-IS traffic before going on the air.
  • Receive is unaffected. You still hear stations, the live map populates, and the logs fill normally.
⚠

Don’t leave simulation on by accident. A digipeater or beacon stuck in simulation is silent on the air even though everything looks healthy in the UI. The dashboard and About page do not currently flag simulation status in a hard-to-miss way — check this page after every config change you intended to ship live.
